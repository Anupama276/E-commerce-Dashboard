# E-commerce-Dashboard
E-Commerce Sales Dashboard This project presents an interactive eCommerce Sales Dashboard designed to visualize year-to-date (YTD) performance metrics for a retail business.Built for strategic insight, the dashboard provides a clear snapshot of overall sales, profit, quantity sold, and profit margin trends across different product categories, states, and customer segments.  
# Key Features:
YTD Metrics: Real-time visualization of Sales, Profit, Quantity, and Profit Margin.
Sales by Category: Breakdown of performance for Office Supplies, Furniture, and Technology.
Regional Insights: Interactive map and pie charts showing sales distribution by state and region.
Product-Level Analysis: Highlights of top 5 and bottom 5 products based on YTD sales.
Shipping Analysis: Comparison of sales by shipping type (Standard, First Class, etc.).
Segment Filters: Toggle between Consumer, Corporate, and Home Office customer segments.
